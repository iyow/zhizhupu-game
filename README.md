## 📖 故事梗概

陈安，25岁城市白领，奶奶在中元节前夕突然去世，回到江南小镇清溪镇奔丧。

老宅里出现的纸人、梦里的亡灵、神秘的纸扎铺老板张老头，以及他那个来历不明的孙女张意——一条条线索引着陈安走进一个维持了六十年的秘密。

原来清溪镇曾有一个执念化成的煞，为了镇压它，两家先人以人魂为锁，代代传承这份沉重的契约。而陈安的出现，是这一切的终点——他是那个煞的血脉，唯一能让它放手的人。

**三大反转：**
- 纸人不只是祭品——有人用它传话
- 那个煞不是外来邪祟——它是陈家祖上的老友
- 奶奶的死，是她主动选择的最后一步棋

## 🎭 角色介绍

### 主角：陈安
- **年龄**：25岁，城市白领
- **性格**：理性、冷静，善于逻辑推理
- **动机**：查明奶奶突然去世的真相
- **隐藏身份**：执念成煞的教书先生的血脉后代

### NPC：张老头
- **身份**：张记纸扎铺老板，第三代守护者
- **性格**：话少，眼神如枯井，知道所有秘密
- **关键台词**："开了光的东西，你就当没看见。"

### NPC：张意（核心人物）
- **外貌**：靛蓝素布裙，竹簪，皮肤白如宣纸
- **性格**：平静，细致，说的每一句都是真话
- **秘密**：颈后朱砂符文，封印着她自己的本质
- **真实身份**：三岁时魂魄被封入纸人，百年后得了宿体——她是活着的

### 已故：爷爷陈志明
- **年代**：1967年
- **牺牲**：捐出一缕魂封印煞，死时不完整
- **梦境现身**：四个字，"出去，快跑"——这是保护

### 反派：煞
- **本质**：教书先生方先生的执念，在寻找失散的家人
- **能力**：模仿人的外貌，侵蚀精神
- **弱点**：血缘认亲，被血脉后代"看见"，执念松动，自然消散

## 🔑 三种结局

| 结局 | 解锁条件 | 内容 |
|------|----------|------|
| **离开** | 理智≥80，希望<40 | 陈安回城市，铜钱留在槐树下，偶尔听见纸张的声音 |
| **留下** | 希望≥60 | 陈安住进老宅，学纸扎，每天早上石阶上有杯热茶 |
| **守护者** | 理智≥80，希望≥80，持有归魂钥匙 | 陈安继承钥匙，成为新的"知情人"，张意说"傻"，没关门 |
