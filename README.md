# 纸扎铺 - 中式恐怖解谜游戏 🏮

> 江南小镇的纸扎铺，传承百年的手艺，却隐藏着不为人知的秘密……

---

## 📖 故事背景

江南小镇的纸扎铺，传承百年的手艺，却隐藏着不为人知的秘密。纸人有时会多出"不该有"的东西，而那些失去亲人的人，会收到来自"阴间"的邀请……

---

## 🎮 游戏特色

- **中式恐怖**：纸人、灯笼、阴宅、中元节等传统恐怖元素
- **多结局分支**：你的选择决定故事走向
- **理智系统**：理智过低会触发坏结局
- **存档系统**：随时保存进度
- **音效系统**：Web Audio API 生成沉浸式音效

## 🚀 快速开始

### 本地运行
直接双击 `index.html` 即可在浏览器中运行！

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 存档 |
| `Ctrl+L` | 读档 |
| `M` | 开关音效 |
| `H` | 显示帮助 |
