## 故事梗概

陈安，25岁城市白领，接到奶奶突然去世的消息，回到江南小镇清溪镇。

老宅里出现的纸人、梦里的亡灵、神秘的纸扎铺老板张老头，以及他那个来历不明的孙女张意——一条条线索引着陈安走进一个维持了六十年的秘密。

原来清溪镇曾有一个执念化成的煞，为了镇压它，两家先人以人魂为锁，代代传承这份沉重的契约。而陈安的出现，是这一切的终点——他是那个煞的血脉，唯一能让它放手的人。

反转：张意并非普通人，而是三岁时魂魄被封入纸人、百年后得了宿体的古老存在。
反转：那个煞不是外来的邪祟，而是陈家祖上的老友，一个死不瞑目的教书先生。
反转：奶奶的死，是她主动选择的最后一步棋——为的是把所有的线索留给陈安。

## 写作风格说明

- **语言**：古韵现代融合，有画面感，不堆砌辞藻
- **节奏**：探索→疑惑→惊悚→解密→反转循环推进
- **恐怖感**：以心理压力和细节氛围为主，非纯粹jump-scare
- **暧昧线**：陈安与张意，以细节行动传达，非直白情感
- **解密层次**：纸人能走路→煞的存在→两家契约→陈家血脉→张意真身
